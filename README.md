# 🎬 Movie Search App

A modern React-based Movie Search Application that allows users to search for movies and view details such as posters, release year, and type using the OMDb API.

![demo](react-movie-search.gif)

##

---

## ✨ Features

* Search movies by title
* Real-time movie data from OMDb API
* Display movie posters
* Show movie title
* Show release year
* Show movie type (Movie, Series, etc.)
* Loading state while fetching data
* Error handling for invalid searches
* Enter key support
* Responsive card-based UI
* Environment variable support for API key security

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
cd react-movie-search
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the project root:

```env
VITE_OMDB_API_KEY=your_api_key
```

Start the development server:

```bash
npm run dev
```
